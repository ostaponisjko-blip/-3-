# Matrix Remote Control 🟢

Локальна система керування звуком через телефон (WebSocket + Node.js)

---

## 🚀 Запуск проєкту

1. Встанови Node.js
2. Відкрий папку проєкту в VS Code
3. В терміналі введи:

```
npm install
npm start
```

Після цього відкрий у браузері:

```
http://localhost:3000
```

---

## 📱 Як підключити телефон

1. Дізнайся IP комп’ютера:

В Windows відкрий командний рядок (CMD) і введи:

```
ipconfig
```

Знайди рядок:

```
IPv4 Address: 192.168.X.X
```

2. На телефоні (в тій самій Wi-Fi мережі) відкрий:

```
http://ТВІЙ_IP:3000/remote.html
```

Приклад:

```
http://192.168.1.5:3000/remote.html
```

---

## 💻 Як підключити ПК

Просто відкрий у браузері:

```
http://ТВІЙ_IP:3000
```
